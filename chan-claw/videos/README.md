## Generate New Videos

```bash
# Using train.js CLI
~/.npm-global/bin/train.js topic "Your Topic" --output my-video.mp4 --theme gradient

# Or use the dedicated OpenClaw video script
~/.npm-global/bin/openclaw-video.js
```

### Available Themes
- dark
- light
- gradient (default)
- corporate
